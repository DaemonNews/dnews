Mailing lists

announce@ this domain is currently the primary mailing list for daemon.news!

Subscribing / Unsubscribing to a list:

Action Address Description
Subscribe <listname>+subscribe@daemon.news Subscribes you to the specified <listname>.
Subscribe Digest <listname>+subscribe-digest@daemon.news Subscribes you to the digest version of specified <listname>.
Unsubscribe <listname>+unsubscribe@daemon.news Unsubscribes you from the specified <listname>.
Help <listname>+help@daemon.news Sends you a brief email describing all the available mailing list commands.

List Netiquette

  1. Don't be a jerk.
  2. Ask well thought-out questions.
  3. Bottom respond (some people still use things like mutt and pine
  4. Advertising is strictly prohibited unless it's for a sanctioned BSD event!
